Introducción

En esta práctica continuaremos utilizando el lenguaje TypeScript para profundizar en sus funcionalidades y diferentes opciones que permite el lenguaje. Esta práctica esta formada por 8 ejercicios que se deberán resolver utilizando lo aprendido en los apuntes de la asignatura y las clases.

Objetivos

Los principales objetivos de esta practica son:

  • Manejo del tipo de datos arrays.
  • Manejo del tipo de datos tuplas.
  • Manejo del tipo de datos enumerados.

Además de los puntos nombrados, se deberá seguir teniendo encuenta lo aprendido en las prácticas anteriores.

Estructura del proyecto

Se ha comenzado la práctica creando la estructura del proyecto como se ha aprendido en las practicas anteriores. Siguiento el tutorial que se nos ha proporcionado. Para cada ejercicio se ha creado un nuevo fichero de código fuente y uno donde se encuentran diversos test.

Para ejecutar los test solo es necesario ejecutar npm test.

Ejercicio 1 (Decodificar resistencias)

Funcionalidad

Se requiere un programa que calcule el valor numérico de una resistencia dependiendo de los colores que tenga esta. Solo se tendrán en cuenta los dos primeros colores.
La tabla que se usará es la siguiente:

Negro: 0 ; Marrón: 1 ; Rojo: 2 ; Naranja: 3 ; Amarillo: 4 ; Verde: 5 ; Azul: 6 ; Violeta: 7 ; Gris: 8 ; Blanco: 9

Código

Para resolver este problema se debe crear una función decodeResistor que recibirá como parámetros los colores en formato String y devolverá un Number con el valor de la resistencia. Para permitir más de dos colores añadiremos como último parámetro un array de String que puede ser pasado o no.

function decodeResistor(res1: string, res2: string, ...res: string[]): number

Para realizar el calculo del valor de cada color, se ha procedido a crear una variable de tipo String que almacena los colores por valor, por lo que cada color estara en la posición del array del calor que tenga. Para evitar errores vamos a pasar a minúscula los parametros de entrada.

const codigos: string[] = ['negro', 'marron', 'rojo', 'naranja', 'amarillo', 'verde', 'azul', 'violeta', 'gris', 'blanco'];

Teniendo el array que tiene la tabla de colores solo debemos buscar para el primer color el valor que le corresponde dependiendo del nombre del color e insertar en una variable de tipo String la posición de este en el array (este primer color pertenece a las decenas).

codigos.forEach((element, index) => {
  if (element === res1) {
    valor += String(index);
  }
});

Ahora que tenemos el primer color hacemos lo mismo con el segundo para añadir el valor de las unidades.

codigos.forEach((element, index) => {
  if (element === res2) {
    valor += String(index);
  }
});

Solo queda tranformar el resultado a Number y devolverlo. return Number(valor).

Ejercicio 2 (Palabras encadenadas)

Funcionalidad

Se requiere un programa que compruebe si un conjunto de palabras estan encadenadas, esto es si una o más letras del final de una cadena coinciden con el comienzo de la siguiente.

Código

Para resolver este problema crearemos una función meshArray, la cual recibirá como parámetro un array de String que contiene las palabas que se desea analizar y devolverá un String con los caracteres que son encadenados o en caso de no serlo devolverá 'Error al encadenar'.

function meshArray(cadenas: string[]): string

Comenzaremos creando una variable de tipo String inicializada a cadena vacía que contendra la solución y una tipo Number para usarla como índice al recorrer las palabras. Se creará un bucle que recorra las palabras del array de entrada, donde almacenaremos la palabra actual y la siguiente a esa, además del índice final de la palabra actual y el inicial de la siguiente palabra.

for (i = 0; i < cadenas.length - 1; i++) {
  const palabraAct: string = cadenas[i];
  const palabraNext: string = cadenas[i + 1]
  let indiceAct: number = palabraAct.length - 1;
  let indiceNext: number = 0;

Haciendo uso de los índices de la palabras, vamos comprobando si no existen caracteres iguales entre la primera palabra y el primer caracter de la siguiente palabra para ir decrementando el índice de la palabra actual. En caso de terminar de recorrer la palabra actual y no encontrar ningun caracter igual, devolveremos la cadena de error.

while (palabraAct[indiceAct] != palabraNext[indiceNext]) {

Una vez encontremos el caracter que coincida, recorreremos desde ese caracter hasta el final de la palabra actual comprobando que para la siguiente palabra tambien va existiendo un caracter igual. En el supuesto de que no concuerden los caracteres se devolvera la cadena de error.

for (let j: number = indiceAct; j < palabraAct.length; j++) {
  if (palabraAct[j] == palabraNext[indiceNext]) {
    sol += palabraAct[j];

Por último se devulve la cadena con la solución return sol.

Ejercicio 3 (Calcular la media y concatenar cadenas)

Funcionalidad

Se requiere un programa que calcule la media de los valores numéricos y concatene los caracteres de un array. El resultado será un array con las dos soluciones.

Código

Para resolver este problema se creará una función meanAndConcatenate que recibirá como parámetro un array con los valores numéricos y caracteres que se desean operar del tipo String o Number y devolverá un array con un String y un Number.

function meanAndConcatenate(cadena: (string | number)[]): (string | number)[]

Necesitaremos dos variables para almacenar cada resultado, la de tipo String la inicializaremos con una cadena vacia y el tipo Number a 0. Para hacer operar correctamente lo que debemos hacer es recorrer el array de entrada y dependiendo de que tipo de dato sea se opera de diferente forma, en caso de que sea String se añade a la variable String y en caso contrario será Number por lo que se sumará al valor anterior y se incrementara en 1 un contador para conocer la cantidad de numeros que se estan procesando.

cadena.forEach((element) => {
  if (typeof(element) === 'string') {
    resultadoCadena += element;
  } else {
    resultadoNumber += element;
    i++;
  }
});

Una vez se ha recorrido todo el array solo queda devolver el resultado como un array donde la primera posición devuelve la cadena concatenada y la segunda posición devuelve la división entre el resultado de la suma de valores y el contador, esto calculará la media de los valores return [resultadoCadena, resultadoNumber / i].

Ejercicio 4 (Mover los ceros al final)

Funcionalidad

Se requiere un programa que envie todos los ceros de un array al final de este manteniendo el orden de los demás elementos.

Código

Para resolver este problema se debe crear una función moveZeros que reciba como parametro un array de tipo Number con los números que se desean ordenar y devolvera un array de tipo Number con el array ordenado como se requiere.

function moveZeros(cadena: number[]): number[]

Primero debemos recorrer el array elemento a elemento, en caso de que no sea 0 no hacemos ninguna operación e incrementamos el índice, pero en caso de si ser 0 lo extraemos del array utilizando el índice e incrementamos la variable que almacena el número de ceros que nos encontramos, no incrementamos el índice ya que al eliminar un elemento ya estamos posicionados en el siguiente.

while (i < cadena.length) {
  if (cadena[i] === 0) {
    cadena.splice(i, 1);
    numZeros++;
  } else {
    i++;
  }
}

Una vez salimos del bucle ya tenemos eliminados todos los ceros, por lo que haciendo uso del número de ceros que encontrados vamos a introducirlos al final.

while (numZeros > 0) {
  cadena.push(0);
  numZeros--;
}

Por último se devuelve el array modificado return cadena.

function multiplyAll(multiplicando: number[]): (multiplicador?: number) => number[]

Lo primero que crearemos será una variable de tipo array de Number donde almacenaremos la solución, seguidamente se devolvera la funcion que realiza el calculo con el parametro de entrada.

return (function multiply(multiplicador: number = 1): number[]

En esta segunda función debemos recorrer todos los elementos del array de entrada, multiplicar cada elemento por el valor que recibió esta segunda función y almacenarlo en la variable que habiamos creado anteriormente.

multiplicando.forEach((element) => {
  sol.push(element * multiplicador);
});

Por último retornaremos el array con la solucion return sol, lo que terminará las dos funciones.

Ejercicio 6 (Puntos bi-dimensionales)

Funcionalidad

Se requiere un programa que permita representar un sistema de dos coordenadas (x, y), en este se deben poder realizar las operaciones:

  • Suma y Resta
  • Producto de un punto por un número
  • Distancia euclidea

Código
  • Sistema de dos coordenadas

Para representar un punto se ha creado un tipo de dato que esta compuesto por dos Number, uno para la coordenada 'x' y otro para la coordenada 'y'.

type punto = [x: number, y: number]
  • Suma y Resta

Para resolver este problema debemos crear la función add que recibe como parámetro dos puntos y devuelve el resultado como otro punto.

function add(puntoA: punto, puntoB: punto): punto

Ahora debemos crear una variable tipo punto para almacenar el resultado, en esta almacenaremos en la primera posicion la suma de los valores de 'x' de los puntos y para la segunda posición almacenaremos la suma de los valores de 'y'.

const puntoC: punto = [puntoA[0] + puntoB[0], puntoA[1] + puntoB[1]]

Por último devolvemos el punto que almacena el resultado return puntoC. Para la operación de la resta se crea otra función substract y se realiza de la misma manera, solo es necesario cambiar la operación '+' por '-'.

const puntoC: punto = [puntoA[0] - puntoB[0], puntoA[1] - puntoB[1]]
  • Producto de un punto por un número

Para resolverlo se debe crear una función multiply que recibe como parámetro un punto y un Number, devolviendo como resultado un punto.

function multiply(puntoA: punto, multiplicador: number): punto

Primero debemos crear una variable tipo punto donde almacenaremos el resultado, la primera posición('x') de esta correspondrá con la multiplicación de la primera posición('x') del punto recibido y el número a multiplicar, la segunda posición('y') correspondrá con la multiplicación de la segunda posición del punto recibido por el número a multiplicar.

const puntoB: punto = [puntoA[0] * multiplicador, puntoA[1] * multiplicador]

Por último devolvemos el punto que tiene el resultado return puntoB.

  • Distancia euclidea

Para resolverlo se debe crear una función euclidean que recibe como parámetros dos puntos con los que se realizará el calculo y devolverá un Number con el resultado.

function euclidean(puntoA: punto, puntoB: punto): number

Creamos una variable que almecene el resultado, que será la raíz cuadrada de la solución de el cuadrado de la resta de la primera posición('x') del segundo punto y la primera posición('x') del primer punto, sumado a el cuadrado de la resta de la segunda posición('y') del segundo punto y la segunda posición('y') del primer punto. Para estor calculor usaremos sqrt para la raíz y pow para el cuadrado.

const sol: number = Math.sqrt(Math.pow(puntoB[0] - puntoA[0], 2) + Math.pow(puntoB[1] - puntoA[1], 2))

Por último solo queda devolver el resultado return sol.

Ejercicio 7

Funcionalidad

Se requiere un programa que permita representar un sistema de 3 o mas dimensiones (x, y, z, ...). En este se deben poder realizar las operaciones:

  • Suma y resta (Deben tener las mismas dimensiones)
  • Producto de un punto por un número
  • Distancia euclídea (Deben tener las mismas dimensiones)

Código
  • Sistema de tres o mas dimensiones

Para representar un punto se ha creado un tipo de dato que esta compuesto por tres o más Number, para permitir que sean 3 o más se establecen 3 Number que deben estar siempre y un último valor que puede estar o no del tipo array de Number.

type puntoMult = [x: number, y: number, z: number, ...dim: number[]]
  • Suma y resta

Para resolver este problema debemos crear la función add que recibe como parámetro dos puntos del tipo puntoMult y devuelve el resultado como otro punto del tipo puntoMult de las mismas dimensiones o un undefined en caso de que no concuerden las dimensiones de los puntos.

function add(puntoA: puntoMult, puntoB: puntoMult): puntoMult | undefined

Comprobamos que el número de dimensiones de los dos puntos es el mismo para poder operar, en caso contrario se devulve undefined.

if (puntoA.length === puntoB.length)

Creamos una variable donde almacenaremos el punto resultante, pero esta no será del tipo puntoMult, sino que será un array de Number, hacemos esto para no preocuparnos por las dimensiones. Para realizar la operación sobre cada coordenada debemos recorrer las coordenadas de uno de los puntos y sumarle la coordenada en la que se encuentre con la coordenada del otro punto con el mismo índice.

puntoA.forEach((element, index) => {
  puntoC.push(element + puntoB[index]);
});

Esto almacena en un array el resultado ordenado, solo debemos devolver el resultado como un tipo puntoMult return puntoC as puntoMult. Para la operación de la resta se crea otra función substract igual que la suma, pero en vez de realizar la suma de las coordenadas lo sustituimos por una resta.

puntoA.forEach((element, index) => {
  puntoC.push(element - puntoB[index]);
});
  • Producto de un punto por un número

Para resolverlo se debe crear una función multiply que recibe como parámetro un puntoMult y un Number, devolviendo como resultado un nuevo puntoMult.

function multiply(puntoA: puntoMult, multiplicador: number): puntoMult

Primero debemos crear una variable tipo array de Number para no tener en cuenta las dimensiones donde almacenaremos el resultado. Para realizar la operación debemos recorrer todas las coordenadas que tenga el punto y multiplicarlas por el numbero, almacenandolo en la variable creada para ello.

puntoA.forEach((element) => {
  puntoB.push(element * multiplicador);
});

Por último devolvemos el array que tiene el resultado pero como tipo de dato puntoMult return puntoB as puntoMult.

  • Distancia euclídea

Para resolverlo se debe crear una función euclidean que recibe como parámetros dos puntosMult con los que se realizará el calculo y devolverá un Number con el resultado o un undefined en caso de que no se pueda realizar el calculo.

function euclidean(puntoA: puntoMult, puntoB: puntoMult): number | undefined

Debemos comprobar que los dos puntos tienen las mismas dimensiones, en caso de ser así continuamos con el calculo, en cualquier otro caso devolvemos undefined.

if (puntoA.length === puntoB.length)

Creamos una variable tipo number que almecene el resultado, en esta añadiremos para cada elemento del primer punto el resultado del cuadrado de la resta de la coordenada actual que concuerde con el índice del segundo punto y la coordenada actual del primer punto (Usaremos pow para el cuadrado).

puntoA.forEach((element, index) => {
  sol += Math.pow(puntoB[index] - element, 2);
});

Por último solo queda devolver la raíz cuadrada del resultado return Math.sqrt(sol).

Ejercicio 8

Funcionalidad

Se requiere realizar un programa que teniendo una coordenada de inicio, una de final y un tablero, calcule el camino para llegar al final utilizarndo los movimientos de Norte, Sur, Este y Oeste.

Código

Para la realización de este problema será necesario importar del ejercicio 6 el tipo de dato punto y la función euclidean que calcula la distancia euclidea de dos puntos.

import {punto, euclidean} from './ejercicio-6'

Lo primero que necesitaremos antes de comenzar con la función principal es una que compruebe si todos los datos recibidos son correctos (si se encuentran dentro del tablero). Para ello crearemos una función inTablero que recibirá un punto del tipo de dato punto, y dos Number con el numero de filas y columnas del tablero, esta devolverá un Boolean con un True en caso de que este dentro del tablero el punto O un false en caso contrario.

function inTablero(puntoA: punto, filas: number, columnas:number): boolean

Comprobamos que las dos coordenadas estan dentro de los limites del tablero, en caso de no estarlo se devuelve false.

Ahora que tenemos un función para comprobar los puntos ya podemos iniciar la función pricipal que calculará el camino agent, esta recibirá como parametro las filas y columnas como tipo Number y el punto inicial y final como tipo punto, devolviendo un array de String con el camino de la solución o un undefined en caso de encontrar algún error.

function agent(filas: number, columnas: number, inicialP:punto, finalP: punto): string[] | undefined

Debemos comprobar que los puntos que se han pasado estan dentro del tablero que se ha delimitado, por lo que llamamos a la funcion creada anteriormente para comprobar cada punto.

if (!inTablero(inicialP, filas, columnas) || !inTablero(finalP, filas, columnas))

En caso de que no esten dentro del tablero debemos terminar el programa devolviendo undefined, pero en caso de que si se encuentren dentro continuamos creando tres variables, una que almacenará en un array de tipo String el camino, la posición actual de tipo punto iniciada con el punto de inicio y por ultimo una tipo Number donde almacenaremos la distancia euclidea de los movimientos que se van comprobando iniciada con la distancia del punto inicial con el final.

const camino: string[] = [];
let posActual: punto = [...inicialP];
let disActual: number = euclidean(inicialP, finalP)

Continuamos creando un bucle que no termine hasta que no se llegue al final (distancia auclidea 0) while (disActual > 0). Existen 4 posibles movimientos: Norte, Sur, Este y Oeste, cada uno se calcula de una forma.

Para calcular el movimiento hacia el norte debemos incrementar en 1 la coordenada 'y'

[posActual[0], posActual[1] + 1]

Para calcular el movimiento hacia el sur debemos decrementar en 1 la coordenada 'y'

posTemp = [posActual[0], posActual[1] - 1]

Para calcular el movimiento hacia el este debemos incrementar en 1 la coordenada 'x'

[posActual[0] + 1, posActual[1]]

Para calcular el movimiento hacia el oeste debemos decrementar en 1 la coordenada 'x'

[posActual[0] - 1, posActual[1]]

Ahora deberemos ir comprobando para cada movimiento si la posicion en la que te deja el movimiento esta dentro del tablero y si la distancia euclidea a disminuido con respecto a la anterior. Para ello creamos una variable que almacene la posible posición para realizar las comprobaciones.

let posTemp: punto = [posActual[0], posActual[1] + 1];
if (inTablero(posTemp, filas, columnas) && euclidean(posTemp, finalP) < disActual)

En caso de que sea valido el movimiento se actualizará la distancia euclídea actual y la posición actual, a parte de introducir en el camino el movimiento que se ha realizado.

disActual = euclidean(posTemp, finalP);
posActual = [...posTemp];
camino.push('North');

En caso de que no sea valido el movimiento o no disminuya la distancia euclídea se comprobara el siguiente movimiento. Una vez se encuentra un movimiento valido se repite el proceso hasta que se llegue al final. Solo queda devolver el array de String con el resultado al salir del bucle return camino.
